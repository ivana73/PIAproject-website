# PIAproject-website

Welcome to the **PIAproject-website** repository! This project is a web application built using TypeScript, CSS, HTML, and JavaScript, NodeJS and Mongodb.

## Language Composition

- **TypeScript:** 44.4%
- **CSS:** 33.1%
- **HTML:** 22%
- **JavaScript:** 0.5%

## Overview

PIAproject-website is designed to deliver an interactive and modern web experience. The project leverages TypeScript for robust, typed development; CSS for responsive, visually engaging interfaces; HTML for structured content; and JavaScript for dynamic behavior.

## Features

- **Type-safe codebase** with TypeScript
- **Responsive design** using modern CSS techniques
- **Structured layouts** with semantic HTML
- **Lightweight JavaScript** enhancements

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/)
- (Optional) A code editor such as [VS Code](https://code.visualstudio.com/)

### Installation

1. Clone this repository:

    ```bash
    git clone https://github.com/ivana73/PIAproject-website.git
    cd PIAproject-website
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Run the project:

    ```bash
    npm start
    ```

### Build

To build the project for production:

```bash
npm run build
```
